# Image Service

Сервис загрузки, хранения и выдачи изображений с модерацией и JWT-авторизацией.

- **Auth API** — регистрация, вход, обновление сессии, профиль: [Swagger UI](http://localhost:8080/auth/swagger/index.html)
- **Internal API** — загрузка, выдача и модерация изображений: [Swagger UI](http://localhost:8080/internal/swagger/index.html)

Запуск: `make up` из каталога `image_service`.
