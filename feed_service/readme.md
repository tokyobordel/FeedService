Сервис обмена изображениями "ИН100ГРАММ".

Назначение: загрузка фотографий пользователями и формирование ленты из постов с изображениям.

Функционал: регистрация и авторизация пользователей, загрузка изображений и формирование с ними постов. 

Ограничения: максимум 2 мб на изображение, максимум три изображения на пост.

Интеграция с сервисами: ImageService (сервис хранения изображений), NotificationService (сервис уведомлений).

Для тестов (создание 500 постов) требуется выполнить ./testPosts.sh

Задание портов:

1. Бекенд: fs_backend.env - BACKEND_PORT, docker-compose.yml - поменять оба порта backend.ports
2. Фронтенд: docker-compose.yml - поменять первый порт в frontend.ports. Урлы внешних сервисов сконфигурировать в .env внутри сервиса

Старт: docker compose up --build
Стоп: docker compose down -v
