# ИН100ГРАММ — фронтенд ленты с постами

Single Page Application для публикации изображений с возможностью регистрации/входа, просмотра общей и персональной лент, загрузки новых постов.

## Структура HTML (`index.html`)

- **`<header class="header">`** — шапка сайта
    - `.logo` — логотип, клик возвращает на общую ленту
    - `#guestButtons` — кнопки «Войти» и «Регистрация» (показываются гостям)
    - `#userBlock` — блок авторизованного пользователя: имя (`#userNameDisplay`), кнопка «Все посты» (`#allPostsBtn`), кнопка выхода (`#btnLogout`)

- **`<main class="main">`** — основная область
    - `#feedContainer` — контейнер для карточек постов
    - `#feedLoader` — индикатор загрузки

- **Модальные окна**
    - `#signinModal` — форма входа (`#signinForm`)
    - `#signupModal` — форма регистрации (`#signupForm`)
    - `#uploadModal` — форма загрузки поста (`#uploadForm`)  
      Все модалки закрываются через `.close`, клик вне окна или вызов `closeModal()`.

- **`<footer class="footer">`** — подвал с копирайтом

- **`#btnUpload`** — плавающая кнопка «+» для открытия формы загрузки (видна только авторизованным)

## CSS-файлы

| Файл | Назначение |
|------|------------|
| `style.css` | CSS-переменные (цвета, тени), сброс стилей, базовые стили body, header, footer, кнопки, адаптив |
| `feed.css` | Сетка постов `.feed`, карточки `.post-card`, слайдер изображений, плавающая кнопка `#btnUpload` |
| `form.css` | Стили форм: группы полей, сообщения об ошибках, список файлов, переключение форм |
| `modal.css` | Модальные окна: скрытие/показ (`display: flex`), фон с блюром, анимация, кнопка закрытия |
| `font-awesome.min.css` | Иконки (Font Awesome) |

## Как работает приложение

1. При загрузке страницы (`DOMContentLoaded`) инициализируются все обработчики (см. `index.js`).
2. Проверяется сессия через `refreshAccessToken()`.
    - Если пользователь сохранён — показывается интерфейс авторизованного (`showLoggedInUI`).
    - Иначе — гостевой (`showGuestUI`).
3. Гость может войти или зарегистрироваться через модальные окна.
4. После входа/регистрации интерфейс обновляется, загружается главная лента (`/api/feed`).
5. Авторизованный пользователь может:
    - кликать на ник автора поста — загружается лента этого пользователя;
    - нажать «Все посты» или логотип — возврат к общей ленте;
    - кликнуть на своё имя в шапке — перейти к своей ленте;
    - загрузить новый пост через кнопку «+» (форма внутри модалки).
6. Слайдер изображений внутри постов переключается кнопками ← → (функции `prevImage`/`nextImage`).
